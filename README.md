# Probability Of Eventual Success

A React + Vite application that simulates the probability of eventual success algebraically and visually. It models various real-world factors that influence success over repeated attempts, such as:
- Base probability per attempt
- Skill growth and exponential learning over time
- Consistency and emotional resilience
- Strategy quality, focus, and adaptability
- Network and environmental advantages

This simulator aims to provide a useful thinking machine to understand how volume and persistence compound with skill and strategy.

## Tech Stack
- React
- Vite
- Tailwind CSS
- Recharts (for data visualization)
- Framer Motion & GSAP (for animations)
- KaTeX (for mathematical formulas)

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Aditya-Bichave/Probability-Of-Eventual-Success.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
